Med2Story-Referential: A Domain-Specific Extension of ISO 24617-9 for Clinical Narrative Annotation

Note: For the best reading experience and proper formatting, please download the documents and view them locally on your device.

Repository Contents: 

  1. Anonymised Annotation Guidelines

  Located in Anonymised_Guidelines.pdf, this document provides a detailed description of the annotation scheme, including the definitions of annotation categories and practical instructions for annotators.

  2. Annotation Results

  The file Annotation_Results.pdf presents both quantitative and qualitative results of the annotation process. It also includes the Inter-Annotator Agreement (IAA) results between ANN1 and ANN2, and as between the annotators and the curator.

  3. Likert Scale Evaluations

  The files ANN1_Likert_Scales.xlsx and ANN2_Likert_Scales.xlsx contain the results of the annotators’ evaluation of the annotation scheme and the guidelines using Likert scales.

  4. Synthetic Medical Reports

  The folder Synthetic_Medical_Reports_docx contains the synthetic medical reports used as the annotation dataset.

Research Context

  This repository accompanies the paper submitted to ISA-22 – The 2026 Joint ACL–ISO Workshop on Interoperable Semantic Annotation, held at LREC 2026.
